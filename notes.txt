
to install latest vim version from mac image app

For bash-language-server on mac:
brew install bash-language-server
sudo cp -r nvim-macos/bin/nvim /usr/local/bin/
sudo cp -r nvim-macos/lib/nvim /usr/local/lib/
sudo cp -r nvim-macos/share /usr/local/

install digestif with wrapper script in:
curl \
  https://raw.githubusercontent.com/astoff/digestif/master/scripts/digestif \
  > digestif_wrap.sh
